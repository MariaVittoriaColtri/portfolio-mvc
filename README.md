## ğŸ¨ Customization

### Adding Images
1. Save images to the `images/` folder
2. Reference them in HTML: `<img src="images/your-image.jpg">`

### Adding Your CV PDF
1. Save your PDF to `assets/Maria_Vittoria_Coltri_CV.pdf`
2. The download button on the CV page will automatically link to it

### Updating Projects
Edit the HTML files in the `projects/` folder to add:
- Real project titles
- Descriptions
- Screenshots
- Links to GitHub repos

### Changing Colors
Edit the CSS variables at the top of `css/style.css`:
```css
:root {
    --primary: #2B488A;        /* Main blue */
    --primary-darkest: #1A2D54; /* Dark blue for headers */
    /* ... etc */
}
```
